# ServoMotor
In this project it's reqired to create a circuit for 5 servo motors to rotate with 90 degrees and control them using potentiometer or varible resistor 

# Components:

- Arduino Uno
- 5 Servo Motors (Micro Servo) 
- Potentiometer (250k Ohm)
